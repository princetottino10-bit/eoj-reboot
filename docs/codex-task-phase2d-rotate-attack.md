## 背景(チーム議論の決定事項)

- フェーズ2cの結果、再命令コストは効いている(構造死ではない)が、安くすると先手勝率が崩れるトレードオフが判明
- チームの方向: **再命令コストはheavy(2/3/3/3/3/4)を軸に**、さらに**攻撃の再命令に回転をセットにする**ルール改定を検証したい
- 追加の仮説(要検証): 「安い再命令で王手を返しても、マナ枯れで結局負けているのでは」(ジリ貧仮説)

## 1. ルール改定(②ルール)の実装

再命令の選択肢を以下の3種にする:

| 選択肢 | 内容 | コスト | 消費する枠 |
|---|---|---|---|
| ① 回転のみ | 90度回転 | 再命令コスト | 回転枠 |
| ② **回転付き攻撃** | **攻撃の直前に90度回転を1回行ってよい**(行わなくてもよい=③と同じ扱い) | 再命令コスト(1回分のみ) | **回転枠と攻撃枠の両方** |
| ③ 攻撃のみ | 従来どおり | 再命令コスト | 攻撃枠 |

裁定(実装上の要点):

- 各式神のターン上限は従来どおり「回転1回まで・攻撃1回まで」。**②はその両方を1アクションで消費する**(②の後に①は不可、①の後に②は不可=回転枠が既に消費されているため。①の後に③は従来どおり可)
- **攻撃の後に回転する手段は存在しない**(③→①は①が回転枠未消費なら可能=現行仕様のまま。これは2回払いなので変更しない)
- ②の回転は攻撃の解決前に行い、攻撃範囲・死角判定は回転後の向きで行う
- 召喚時攻撃には適用しない(召喚時は向きを自由に選べるため不要)
- AI・チェック返答探索・オラクルの行動空間に②を追加すること(これを忘れると計測が無意味になる)

## 2. 計測の追加(全ランに適用)

### 勝敗別の再命令使用(ジリ貧仮説の検証)

- **勝者側/敗者側それぞれ**の、攻撃再命令回数・回転回数・②(回転付き攻撃)回数/試合
- **王手を返したプレイヤーのその試合の勝率**(返答成功がその後の勝ちに繋がっているか。仮説が正しければ、安い水準ほど「返したのに負ける」が増えるはず)

### 既存計測

フェーズ2cと同じ全KPI+計測B/C/D(②は攻撃系としてB/C/Dに含め、②の使用は攻撃と別カウントでも報告)。

## 3. 実行(2×2要因+参考2水準、各2000試合)

すべて seed `20260702`、oracle あり、depth 6、その他はv3構成固定:

| ラン | 再命令コスト | ②ルール | 目的 |
|---|---|---|---|
| R0a | heavy+(現行 2/3/3/4/4/5) | なし | 対照(=v3。新計測のため再実行) |
| R0b | heavy(2/3/3/3/3/4) | なし | コスト効果の対照(新計測のため再実行) |
| R1 | heavy | **あり** | **本命**(チームの想定構成) |
| R2 | heavy+ | あり | ルール効果の分離(コストを変えずにルールだけ入れた場合) |
| R3 | standard(2/2/2/3/3/3) | なし | ジリ貧仮説の検証用(勝敗別計測つき再実行) |
| R4 | light(1/2/2/2/2/3) | なし | 同上 |
| R5 | standard | なし・**DESTROY_MANA_GAIN=2** | standard採用パスの探索: dm2で先手勝率47%を帯内(50〜53%)に戻せるか |
| R6 | standard | **あり**・DESTROY_MANA_GAIN=2 | 同上+②ルール併用時 |

R0a〜R6は同一シード系列(フェーズ2cと同じペア比較方式)。R5/R6以外はすべて DESTROY_MANA_GAIN=1。

R5/R6の背景: フェーズ2aで「後手マナとDESTROY_MANA_GAINは相殺関係(dm+1≒後手マナ+1の打ち消し)」が実測済み。standardの先手勝率47%を、後手+1(初手パス問題のケア)を維持したままdm側で先手に半段寄せる仮説の検証。

## 4. 特に見たい判定材料

1. **R1(heavy+②)の先手勝率**: ②ルールで攻撃行動が増える分、先手有利/後手有利がどちらに動くか(チームの懸念事項)
2. R1の回転関連: ②の使用回数と、「王手返しで②が必須になるケース」(計測Dに②除外判定を追加: ②を禁止した制限オラクルでも返せるか)
3. ジリ貧仮説: R3/R4(安い水準)で「王手を返したプレイヤーの勝率」がR0a/R0bより低いか、敗者側の再命令使用が勝者側より多いか
4. R1の全KPIが帯内か(平均R 7〜10、返し率45〜55%、撃破3.5〜6など)
