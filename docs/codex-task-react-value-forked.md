# Codex Task: 再命令の損得計測(強制ON/OFF分岐、R1構成)

Status: 実行待ち
Date: 2026-07-08
納期メモ: 7/18の共有資料に使用予定

前提資料(先に読むこと):

- `docs/baselines/phase2d-rotate-attack.md` — R1構成の定義と計測
- `docs/codex-task-v31-freeze-and-items.md` — 採用済み構成(v3.1=R1)の正確な仕様

## 目的(知りたいことは一点)

**「R1(再命令コストheavy+回転攻撃ルール)の値段でも、攻撃系再命令は"使うのが正解"になっているか」**を、状況別の因果比較で判定する。

- 使うのが正解 → コスト設計はこのまま完成
- 使わない方が勝つ局面が支配的 → コストは触らない(確定済み)ので、フェーズ4で再命令側に価値を足す(割引霊具・攻撃報酬効果など)必要があると事前に分かる

## 構成(固定)

R1 = v3.1: 再命令コストheavy(2/3/3/3/3/4)+回転攻撃ルール(②)。式神16枚のみ(霊具なし)、他はv3構成(hybrid/high/lowmid、先手3/後手4、dm1)。

## 方法: 強制ON/OFF分岐(カウンターファクチュアル比較)

### 分岐プロトコル

1. 通常のR1ミラー対戦を進行し、**「攻撃系再命令が使用可能な局面」**を検出する。定義: 手番プレイヤーに、霊力が足りて、攻撃対象が存在する ③攻撃 または ②回転攻撃 の合法手が1つ以上ある(①回転のみは対象外)
2. 対象局面から**同一状態でゲームを2つに分岐**し、両方とも最後まで走らせる:
   - **ON世界**: 最良の攻撃系再命令(既存の行動スコアで選択)を1回強制実行 → 以降は通常AIで続行
   - **OFF世界**: そのターン中、攻撃系再命令(②③)を禁止(①と召喚・その他は通常どおり) → 以降のターンは通常AIで続行
3. 分岐は**1試合につき最大2局面**(該当ターンから一様サンプリング)。分岐しすぎによる計算爆発を防ぐ
4. 分岐後の継続は両世界で同じ乱数系列を使う(ペア比較)

### 層別(必須 — 効果の異質性を見る)

各分岐局面を、分岐時点の状態で分類:

- **優劣**: 自分の占拠数 − 相手の占拠数 → 劣勢(≤−1)/互角(0)/優勢(≥+1)
- **王手状況**: 王手被弾中(相手の占拠≥4)/通常

2×3=6層(疎な層はnを明記してそのまま報告)。

### 行動種別の内訳(必須)

- ON世界で強制実行した手が**②か③か**を記録し、損得を「②+③合算」と「②のみ/③のみ」の内訳で報告(②ルール自体が損得に効いているかを見るため)
- 任意(可能なら): ②が最良と選ばれた局面のうち**③も合法な局面**に限り、第3の世界「ON-③(回転なしで同じ式神が攻撃)」も走らせ、②と③の直接比較を出す

## 計測(層×行動種別ごと)

| 指標 | 内容 |
|---|---|
| n | 分岐ペア数 |
| ON勝率 / OFF勝率 / **Δ(ON−OFF)** | 主指標。Δ>0なら「使うのが正解」 |
| 平均決着ラウンド(ON/OFF) | 使用が試合を縮めるか伸ばすか |
| 通常AIの使用率 | その局面で(強制しない場合)AIが自発的に使っていた割合。AIの判断と因果の答え合わせ |

## 実行

- ベース3000試合(分岐ペアを層別に十分確保するため)、seed `20260702`、depth 6
- 分岐世界の対戦もすべて完走させる(推定: 3000×2局面×2世界=最大12000部分試合)
- オラクル計測は本タスクでは不要(計算節約のため省略可。チェック返答探索は通常どおり)

## フォールバック

分岐実装が困難な場合のみ: 観測ログの層別比較(「使ったターン」vs「使えたが使わなかったターン」を同じ6層で比較)。ただし**この方法は交絡が残る**(同じ層内でもAIが使う局面は使わない局面と系統的に違う)ため、レポートに必ずその限界を明記すること。
