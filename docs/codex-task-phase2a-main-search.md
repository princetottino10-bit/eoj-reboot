## 1. スコア関数の更新

`scoreSummary` を「確定KPI帯」に完全準拠させる:

- 平均R 7〜10(帯外にペナルティ)、P90≤14、占拠≥95%、引き分け0%、先手50〜53%、4チェック返し45〜55%(**上限超過も減点**)、撃破3.5〜6
- 攻撃再行動2.5〜4.5、弱点撃破率≥45%
- 手詰まり≤20%、マナ利用率≥87%、盤面3体到達 中央値≤4R
- **回転関連はスコアに含めない**(ログのみ)

## 2. グリッド(一次選抜)

カーブ8種 × 分布4種 = 32変種、ミラー対戦、各1000試合、seed `20260702`、oracle あり、check-search-depth 6。

カーブ(HP × ATK × 再行動):

| 軸 | 候補 |
|---|---|
| HP | lean: 2/3/4/5/6/7、hybrid: 2/3/4/6/7/8(低コストlean+高コストstandard) |
| ATK | high: 1/2/2/3/3/4、standard: 1/1/2/2/3/3 |
| 再行動 | heavy: 2/3/3/4/4/5、standard: 2/2/3/3/4/4 |

(キャリブレーションv2でHP standardの返し率・キルが劣化したため、lean系を中心に再構成。react lightは後手有利が過大なため除外)

## 3. 上位候補の再校正(二次)
