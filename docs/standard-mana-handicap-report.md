## 1. 何を調べたか

再命令コストstandard(2/2/2/3/3/3)は使用回数が良い(3.25回/試合)一方、先手勝率46.9%(後手有利)が採用のネックだった。フェーズ2dでは `DESTROY_MANA_GAIN=2` による補正を試したが、「dmはゲームプレイ上の意味を持つルールなので、バランス調整のノブにはしない。**調整は純粋な調整弁である初期霊力ハンデで行う**」という方針に改め、ハンデの掃引で帯内(50〜53%)に収まる構成を探した。

- 現行の初期霊力: 先手3 / 後手4(後手+1)
- 掃引: 後手+0 / +1 / +2 の3点、②ルール(回転付き攻撃)の有無を併せて5構成
- 各2000試合、同一シード系列、dm=1固定、他はv3構成固定

## 2. 結果

### その他のKPI(要点)

- 平均R 7.9〜8.3、返し率53.4〜54.3%、撃破4.8〜5.5 — どの構成もこの辺りは健全
- ②あり構成(S2/S3)は攻撃系使用4.45〜4.49と帯上限(4.5)に張り付き、返し手勝率も37〜38%まで低下(2dのR6と同じ「動きすぎ」傾向)

## 3. 結論

**standardを帯内に収める構成は、承認済みの調整ノブ(初期霊力ハンデ)の範囲には存在しない。**

- ハンデは整数刻みでしか動かせず、1段の感度(約9pt)が帯の幅(3pt)より大きいため、帯が段差の間にすっぽり落ちる
- ②ルールの併用でも橋は架からない(S2: 54.9% / S3: 45.2%)
- これによりフェーズ2c以降の再命令コスト探索は完結: light/standard系はすべて先手後手バランスで脱落し、**heavy+②ルール(フェーズ2dのR1: 先手50.0%・攻撃系使用3.47・全KPI帯内)が唯一の帯内完走構成**として残った

## 4. 採用判定に向けて(保留中)

- 残る選択肢は実質「R1(heavy+②)採用」か「現行維持(heavy+、使用2.36は帯下端割れのまま)」の2つ
- R1採用時の後続作業: RULE_SPEC §9.3の改訂(②ルールの条文化)→ 基準v3.1として凍結 → 霊具レイヤーの再実行
- 次回ランでの計測追加(予約済み): 「4チェックの返答に失敗したプレイヤーの勝率」— 返し手勝率(成功側)との対比で「返す価値」を直接示すため
