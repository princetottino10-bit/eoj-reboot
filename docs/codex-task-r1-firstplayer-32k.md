# Codex Task: R1構成の先手勝率 高精度測定(3.2万試合)

Status: 実行待ち
Date: 2026-07-12

前提資料: `docs/baselines/phase2d-rotate-attack.md`(R1構成の定義)

## 目的

採用済みR1構成の**先手勝率だけ**を高精度で確定させる。従来レポートは2000試合(95%CI ±2.2pt)で、目標帯(50〜53%)の幅3ptに対して分解能が不足していた。32,000試合(95%CI ±0.55pt)で測り直す。

## 構成(固定・変更禁止)

R1 = v3.1: 再命令コストheavy(2/3/3/3/3/4)+回転攻撃ルール(②)、式神16枚のみ(霊具なし)、HP hybrid / ATK high / lowmid分布、先手霊力3/後手4、dm1、MAX_MANA 15。

ルール裁定(継続): 回転攻撃は回転枠+攻撃枠を消費 / 撃破判定で撃破された式神は反撃しない / 墓地行きで所有者がdm1獲得。

## 実行

- ミラー対戦 **32,000試合**、seed `20260702`(既存のシード系列 `seed + gameIndex * 9176` をそのまま32,000まで延長)

## 計測・報告

1. **先手勝率(全体)**: 点推定+95%信頼区間(±1.96×√(p(1−p)/n))
2. **2000試合ごとのバッチ別先手勝率(16バッチ)**: バッチ間のばらつきを表で示す。「2000試合測定がどれだけ揺れるか」の実証として、既存レポート群の読み方の校正材料になる
3. 参考値としてついでに取れるもの(平均R、占拠勝ち率、引き分け率)は1行で併記してよいが、レポートの主題は先手勝率のみ
