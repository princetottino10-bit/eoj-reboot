# CC Handoff: 7/18 Print Kit

更新日: 2026-07-17

## 現在の結論

7/18紙プレイテスト用の印刷キットは、印刷仕様との照合、同仕様での2000試合確認、PDF全5ページの目視確認まで完了した。

- カードサイズ: 63 x 88 mm
- A4 1枚に3 x 3の9枚配置
- 1人16枚 x ミラー2人分 = 32枚
- 9種: 鬼火4 / 小鬼4 / 槍霊1 / 薙刀霊1 / 鬼2 / 両面1 / 仁王1 / 九尾1 / 龍1
- 各プレイヤーに地・水・火・風を4枚ずつ固定割り当て
- 攻撃範囲と反撃範囲は同一
- 管理ID (`7/18 onibi-P1-1` など) はカード面から削除済み
- 旧誤表記「斧兵」は「薙刀霊」へ修正済み

## 印刷仕様

- C2/C3/C4のHPは3/4/5
- 九尾は斜め前2・前1・左右の5マス
- 範囲図は5 x 5で、攻撃・反撃、弱点、自分の位置を同時表示
- 生命価はC2-C4が1、C5-C6が3、C7が4
- 龍の再命令コストは4、それ以外はコスト帯表どおり

HTML照合は `scripts/validate-print-kit-0718.mjs` で32枚すべてPASS。結果は `docs/print-kit-0718-audit.md` に保存した。
