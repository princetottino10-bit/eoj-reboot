## エンジン修正(RULE_SPEC準拠)

1. **デッキ切れ**: ファティーグを撤去。デッキが空になったら墓地をシャッフルして新しいデッキにする(§15)。デッキ切れ敗北なし
2. **回転回数**: 現行のまま**1式神1ターン1回**(裁定によりRULE_SPEC側を改訂予定。変更不要、確認のみ)
3. **召喚位置制限**(§9.2): 実装状況を確認し、なければ実装:
   - 自分の式神が盤面に1体もいない → 任意の空きマス
   - 自分の式神が存在する → いずれかの式神(敵味方問わず)と上下左右で隣接する空きマスのみ
   - AI・オラクル・チェック探索の召喚候補生成にも同じ制限を適用
4. **撃破時の霊力補償**(§9.5): 式神が**盤面から墓地へ送られたとき**(戦闘撃破・効果破壊・生贄を問わず)、その所有者は `DESTROY_MANA_GAIN` を獲得。**暫定値1**(フェーズ2で感度測定)。獲得量を集計に追加(裁定済み: 原因を問わず墓地行きで獲得)
   - あわせて戦闘順序の裁定を実装に反映: **撃破判定(§10 手順4)で撃破された式神は、反撃(手順5)を行わない**
5. **属性体系**(§11): 地/水/火/風/空 の5属性に変更。対立は 火↔水、地↔風。空は対立なし
   - 純粋素体への割当: 地/水/火/風 を均等ランダム(空は素体に割り当てない)
6. **盤面属性配置**(§7): 推奨配置で固定:
   ```
   水 火 風
   地 空 水
   火 風 地
   ```
   (空マスは属性修正が誰にも発生しない)
7. **属性HP修正**(§11.2): 「マスにいる間、現在HPに適用・解除」方式に変更。修正の適用・解除でHPが0以下になったら撃破判定。純粋バニラでは移動効果がないため実質は召喚時適用と同じだが、実装は仕様準拠の形にしておく
8. **HP上限**(§9): `MAX_HP` を実装。**暫定値10**(カーブ最大8+属性+2で、純粋バニラでは挙動に影響しない位置。フェーズ2で正式決定)
9. **生命価**(§9.5): コスト帯定数(life134)をカード側ステータス「生命価」に変更。純粋素体への付与カーブ: C2〜C4=1 / C5〜C6=3 / C7=4(数値は従来と同一)
10. **霊力補充タイミング**(§6.4): ターン開始時→**自ターン終了時**に移動。ターン進行は §6 の順(開始→メイン→終了→霊力補充→手札補充)
    - これに伴い初期霊力を調整: `START_MANA_FIRST = 3` / `START_MANA_SECOND = 4`(従来の「開始時+3、後手+1」と同じ初動経済を保つ暫定値)
11. **霊力上限**: `MAX_MANA = 15` を維持(裁定により正式採用)
