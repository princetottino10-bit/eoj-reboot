## 1. 何を調べたか

採用済みR1構成(再命令コストheavy+回転攻撃ルール、式神16枚=7/18テストと同一構成)の**先手勝率**を高精度で確定させる。

背景: これまでの全レポートは2000試合測定で、95%信頼区間は±2.2pt。目標帯(50〜53%)の幅3ptに対して分解能が足りず、「48.4%と50.0%の差は誤差か実力か」が判別できなかった。32,000試合(±0.55pt)で決着させる。

## 2. 結果

| 指標 | 値 |
|---|---|
| **先手勝率** | **50.125%** |
| 95%信頼区間 | 49.58〜50.67% |
| 試合数 | 32,000 |

**設計理想の50%と統計的に区別できない。** 先手・後手の公平性の調整(後手の初期霊力+1 × 撃破時霊力獲得)は、この構成において完成している。

## 3. 副産物: 「2000試合測定の揺れ幅」の実証

32,000試合を2000試合ずつの16バッチに分けた先手勝率:

| 指標 | 値 |
|---|---|
| バッチ最小〜最大 | **47.95%〜51.90%** |
| レンジ幅 | 3.95pt |

同一条件・同一エンジンでも、2000試合の測定は**±2pt程度揺れる**ことが実証された。これは事前の見立て(確率収束がそのくらい)どおり。

**過去レポートの読み方への影響**: これまでの2000試合測定で出た先手勝率のばらつき(48.4%、49.3%、50.0%など)は、すべてこの揺れ幅の中にある。**過去の構成間で±2pt以内の先手勝率差を根拠にした優劣判断は無効**であり、今後、先手勝率を判定基準にする場合は本レポート同様の大試行数(または±2ptを超える差)を要件とする。

なお、過去の採用判断(R1採用など)は先手勝率だけでなく複数KPIの総合で行っているため、結論への影響はない。lightやstandardの47%割れ(帯外判定)は揺れ幅を考慮しても境界〜外側であり、コスト水準間の傾向(安いほど後手有利)は複数水準の単調性で支えられているため、こちらも維持される。

## 4. 含意

1. **R1構成の先手勝率は50.1%で確定。** 7/18テストパッケージの期待値表は「約50%(3.2万試合)」に更新
2. **先後公平の2つの重り(後手霊力+1、撃破霊力1)はこれ以上触る必要なし。** 今後レイヤー(霊具・効果)を載せて先手勝率が動いた場合のみ再調整
3. **測定プロトコルの改訂**: 先手勝率を合否判定に使うランは、今後2000試合では不足。①判定用は8000試合以上(±1.1pt)を推奨、②探索・比較用は2000試合のままでよいが±2ptの揺れを常に添えて読む
