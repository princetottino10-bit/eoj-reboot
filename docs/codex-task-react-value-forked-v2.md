## v1のどこが壊れていたか(必読)

v1の−20.7ptは「再命令が損」を意味しない。2つのアーティファクトが乗っていた:

1. **比較が非対称だった**: ONは「1回強制実行(AIの意思を上書き)」、OFFは「1手禁止した上でAIが残りから再最適化」。制約下で選び直せるOFFが構造的に有利で、その証拠にミラー戦なのにOFF勝率が53.8%(=50%であるべき)。
2. **全局面で強制していた**: 分岐点はAIが本来なら67%は再命令しなかった局面を含む。「AIが使わないと判断した局面で無理に使わせれば損」という自明な結果を主に測っていた。層別でAI自発使用率が低い層ほどΔが悪い(互角×通常: AI使用11%でΔ−24pt)のがその裏付け。

## 修正方針: 「AIが実際に使った局面」で対称に比較する

つなけんさんの原問「再命令を**使えた局面**で、使った場合と使わなかった場合、その後どっちが勝つか」を、バイアスを除いて実装する。

### 分岐トリガー(v1から変更)

**通常AIが実際に攻撃系再命令(②回転攻撃 または ③攻撃)を選択した局面のみ**を分岐点とする。v1の「合法手が存在する局面すべて」ではない。これにより「AIが使いたいと判断した場面での損得」に問いを絞る(="使えた局面で使うのは正解か"の直接の答え)。

### 2世界の定義(対称化)

- **ON世界(=AI通り)**: AIが選んだその攻撃系再命令を実行 → 以降ずっと通常AI
- **OFF世界(=その1手を回避)**: その手番で、**その攻撃系再命令の代わりにAIの次善手を実行**(攻撃系再命令②③を除いた行動集合でAIに選び直させる。①回転・召喚・待機などが候補) → 以降ずっと通常AI

両世界とも「1手だけ差し替え、以降は同じ通常AI」。ONも「AIが選んだ手をそのまま実行」なので、両者とも意思の上書きをしていない(ONは追認、OFFは第2希望)。これで非対称が消える。

### 対称性の検証(必須・実装の正しさチェック)

- 全分岐ペアの単純平均で **ON勝率とOFF勝率の合計が約100%** になること、かつ**分岐対象プレイヤーの無条件勝率が約50%付近**になることをレポート冒頭で確認する。もしOFFが明らかに50%から外れる(vのように53%等)なら実装がまだ非対称なので、その旨を報告して停止。

## 層別(v1と同じ)

分岐時点の状態で分類:
- 占拠差: 劣勢(≤−1)/互角(0)/優勢(≥+1)
- 王手被弾中(相手占拠≥4)/通常
- 2×3=6層、疎な層はn明記

## 行動種別

②③合算に加え、②のみ/③のみの内訳(AIが②を選んだ局面/③を選んだ局面で別集計)。

## 計測(層×種別)

| 指標 | 内容 |
|---|---|
| n | 分岐ペア数(=AIが攻撃系再命令を選んだ局面のサンプル数) |
| ON勝率 / OFF勝率 / **Δ(ON−OFF)** | 主指標。Δ>0=「AIの判断(使う)が正しい」 |
| 平均決着ラウンド(ON/OFF) | |
| OFFで代わりに選ばれた手の内訳 | 召喚/①回転/待機など。「使わない場合の最善代替」が何かを見る |
