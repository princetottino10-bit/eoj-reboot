# 効果設計ガイドライン(ループ対策と値付けの原則)

Status: 草案(レビュー待ち)
Date: 2026-07-05
対象: 検討事項トラッカー #9

フェーズ4(式神効果・実霊具の値付け)で効果を設計・改訂する際の原則集。背景にある構造的リスクは「**墓地リシャッフルにより資源が無限**」であること——デッキ切れがない以上、繰り返し可能な純増ループは必ず無限ループになる。

## 原則1: 霊力の純増ループを作らない

任意の「繰り返し可能な行動列」について、霊力収支が純増にならないこと。

- チェック式: 効果1回の霊力生成 ≤ 発動コスト + カード1枚分の機会費用(目安: 霊力1)
- 例: 「霊力を2得る」霊具のコストは1が下限(正味+1で、カード消費が歯止め)。コスト0で霊力+2は、リサイクル手段と組んだ瞬間に無限化する
- **生贄・自壊系は `DESTROY_MANA_GAIN=1` を収支に含めて計算する**こと。「味方1体を破壊して霊力Xを得る」の実質獲得は X+1

## 原則2: ドローの純増にもコストを課す

墓地リシャッフルによりデッキは無限に回るため、「霊力1未満でカード1枚を引ける」効果はデッキ全体を1ターンで掘れる装置になりうる。

- 目安: ドロー1枚あたり実質コスト1以上を維持
- 「引いたカードのコストを踏み倒す」系は別枠で厳しく(踏み倒し=霊力生成と等価)

## 原則3: 効果のコピー・再発動は入れ子にしない

召喚時効果の再発動・コピー系(旧プールの連環の指揮者、写し鏡の双子、鎖の解放者の声など)は連鎖の温床。

- 標準裁定: **「効果を発動する効果」はコピー・再発動の対象にできない**(コピーのコピー禁止)
- 再発動系の効果には回数制限(1ターンN回)を標準装備する

## 原則4: 例外は基本ルールでなくカードに書く(裁定済み方針)

問題のある循環が見つかったら、基本ルール(RULE_SPEC)に例外を足すのではなく、該当カードのテキストで塞ぐ。

- 定型句の例: 「この効果による破壊では霊力を獲得しない」「この効果は1ターンに1回しか発動しない」
- RULE_SPEC §18(カードテキスト優先)と整合

## 原則5: 「1ターンにN回まで」を標準の設計部品にする

旧プールには既に多数の回数制限つき効果がある(移ろいの祠、略奪の狼煙など)。これは正しい習慣なので、繰り返し発動しうるトリガー効果(〜するたび系)には**デフォルトで回数制限を付け、外す場合に理由を要求する**運用にする。

## 原則6: 値付けは1変数ずつ、素体差分で測る

- 効果の値段 = 「素体+この効果だけ」のシミュ勝率が素体と釣り合うコスト増分
- 複数の効果を同時に足して測らない(交絡)
- 効果込みで無限ループの検出ランを回す(同一盤面ハッシュの再訪検出は引き分け判定§16の実装としても必要)

## 設計時チェックリスト

新しい効果を書いたら以下を確認:

- [ ] 繰り返し可能な行動列で霊力が純増しないか(dm=1込みで計算)
- [ ] ドロー1枚あたり実質コスト1以上か
- [ ] 効果の再発動・コピーの入れ子が発生しないか
- [ ] 「〜するたび」系に回数制限があるか(ない場合、理由を明記したか)
- [ ] 墓地リシャッフルと組んだときに挙動が変わらないか(墓地枚数参照・墓地回収系は特に)
- [ ] 秘術・生命価との相互作用(生命を回復/支払う効果は敗北条件に触る)
