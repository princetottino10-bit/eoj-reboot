# Codex Task: 7/18形状構成 もう1段トリム(Tバッチ)

Status: 実行待ち

納期メモ: 7/18最終構成の決定に使用(至急)

前提資料: `docs/baselines/shapes-0718-m.md`(Mバッチ)。**反撃範囲=攻撃範囲で確定**(M1a系)。

## 背景と狙い

M1a(トリム済み形状+反撃=攻撃範囲)は平均11.69R・占拠勝ち87.1%・生命勝ち12.8%で、目標(9〜10.5R・生命勝ち≤10%)にまだ届かない。Mバッチ分析より、**乱戦の主因は広い駒ではなく低コスト帯(足軽・剣士)の隣接殴り合い**(剣士だけで撃破・被撃破が突出)。今回は低コスト帯の殴り合い密度を下げる2案を当てる。全ラン**反撃範囲=攻撃範囲**。

## デッキ基準

Mデッキ(足軽4/剣士4/弓兵1/斧兵1/重剣2/双頭1/大剣1/旋風1/竜1)を出発点とする。重剣=C4前1・弱点後1、大剣=C5前1・弱点後1。座標は前=+x、attack_mode=choice、物理、反撃=攻撃範囲。

## 実行(2案×2000試合、seed 20260702、oracleあり、depth 6)

### T1: 低コスト枚数の暴れを抑える
Mデッキの**剣士4→2、その分 重剣2→4**(前1・弱点後1の詰め物を増やし、攻撃的な素の殴り合いを減らす)。他は同じ。

デッキ: 足軽4 / 剣士2 / 重剣4 / 弓兵1 / 斧兵1 / 双頭1 / 大剣1 / 旋風1 / 竜1 = 16枚

### T2: 低コスト帯のHP+1
Mデッキそのまま(枚数不変)。ただし**コスト2・3の式神のHPを+1**する(C2: 2→3、C3: 3→4)。「1発で落ちない」ことで撃破連鎖を抑え、生命勝ちを直接下げる狙い。C4以上は変更なし。

## 計測・報告

- 形状別内訳(撃破/被撃破/③/②)を T1/T2 について
- 参考目標(判定は人間側): 平均9〜10.5R、生命勝ち≤10%、占拠勝ち≥90%、先手勝率は±2.2pt注記つきの参考値
