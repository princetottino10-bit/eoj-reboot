## 変更内容(7/18印刷仕様からの差分)

ベース: 7/18印刷仕様(9種16枚デッキ・形状・反撃範囲=攻撃範囲・再命令heavy+回転攻撃)。そこに以下を適用:

### ルール変更

1. **撃破時霊力: 1 → 2**(盤面→墓地で所有者が霊力2獲得)
2. **配置の隣接制(現行ルール維持を確認)**: 自分の式神が盤面に1体もなければ任意の空きマス。1体でもあれば、いずれかの式神(敵味方問わず)と上下左右に隣接する空きマスにのみ召喚可。**現行ルールから変更なし**(確定事項としての再確認)
3. **勝利条件(現行ルールの明文化・変更なし)**: 5体目を召喚し、召喚時の処理(召喚攻撃・反撃・召喚時効果)がすべて解決した後、**自ターン終了時に自分の式神が5体いれば勝利**。5体目が召喚攻撃への反撃で撃破された場合は不成立(4体に戻る)。判定タイミングは現行の「自ターン終了時に5マス占拠」と同一であり、**エンジン・チェック探索の改修は不要**。生命0勝利も従来どおり
4. **死角(弱点)ボーナス: +2 → +1**(反撃不可は継続)

### ステータス変更

5. **C2/C3のHPを印刷仕様から−1**: C2 HP 3→2、C3 HP 4→3(=v3.1素体値に戻る)。C4はHP5のまま、C5以上も変更なし

### 属性システムの変更(大)

6. **属性を陰陽2属性に変更**: 各プレイヤーの16枚に陰8・陽8をランダム割当(従来の4属性均等割当の置き換え)
7. **盤面配置(固定)**:
   ```
   空   陰   空
   陰  太極  陽
   空   陽   空
   ```
   - 空(四隅)=属性修正なし / 陰マス・陽マス=下記修正 / 太極(中央)=下記効果
8. **属性ボーナス±2**: 同属性マスでHP+2、対立属性マス(陰↔陽)でHP−2。現在HPに適用・離れると解除・HP上限10は従来どおり
9. **太極の効果**: 太極マスに自分の式神がいる間、**その所有者が支払う召喚コストを1軽減(下限1)**。**再命令コストは軽減しない**。属性修正はなし。どちらのプレイヤーでも同条件

### 解釈メモ(実装時に迷ったら)

- 太極の軽減は召喚のみ(1ターン1召喚なので実質ターン1回・−1)
- 陰陽の割当はコスト帯と独立にランダム(シード管理下)
- 上記解釈がユーザー意図とズレている可能性があるため、実装ノートに採用した解釈を明記すること
