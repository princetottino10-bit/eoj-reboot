## 1. 何を調べたか

フェーズ2c(再命令コスト掃引)の結果を受けたチーム議論から出た2つの案と1つの仮説を検証した。

**案1: 回転付き攻撃(②ルール)** — 再命令の選択肢を「①回転のみ / ②回転してから攻撃 / ③攻撃のみ」の3種にする。②は攻撃の直前に90度回転を1回行ってよく、コストは再命令1回分のみ。**攻撃の後の回転は認めない**(攻撃後に死角を壁側へ向けるフォローが定石化すると、ハサミ打ち等の位置取りの読み合いが消えるため)。

- 裁定: ②は回転枠と攻撃枠の両方を消費する(各式神のターン上限「回転1回+攻撃1回」は不変。1ターンに90度までしか向きを変えられない原則を維持)

**案2: standard採用の可能性** — コストstandard(2/2/2/3/3/3)は使用回数が良い(3.27回)が先手勝率46.9%が難点だった。後手霊力+1は維持したまま(初手パス問題のケアとして)、`DESTROY_MANA_GAIN=2` で先手側に半段寄せて帯内に戻せるか。

**仮説: ジリ貧仮説** — 「安い再命令で王手を返しても、そのマナ消費で結局ジリ貧になって負けているのでは」。検証のため**勝者側/敗者側別の再命令使用回数**と**「王手を返したプレイヤーのその試合の勝率」**を新規計測。

## 4. 主要な発見

### 発見1: R1(heavy+②)は全KPIが帯内。先手勝率は全探索史上初の50.0%

②ルールの狙い——「standard並みの使用回数を、heavyの先手後手公平さのまま手に入れる」——がそのまま実現した:

- 攻撃系再命令 3.47/試合(帯2.5〜4.5の中央付近。現行2.38の1.5倍)
- 先手勝率 50.0%(帯50〜53%)
- 平均8.19R、返し率52.2%、撃破4.9、手詰まり19.3% — すべて帯内
- **②でしか返せない王手が0.10件/試合存在する**(制限オラクル判定)。「回転してから斬る」は飾りではなく固有の戦術として機能している

### 発見2: standard採用パス(R5/R6)は惜しいが帯の縁に乗り切らない

- R5(standard+dm2): 先手勝率46.9%→53.6%。dm2の補正(+6.7pt)は効いたが、**逆側にわずかに帯超え**。standardのコストは「補正なしでも補正ありでも帯に入らない谷間」に落ちる
- R6(standard+dm2+②): 先手52.9%は帯内だが、攻撃系4.54(帯上限超)・返し率55.3%(上限超)と全体に動きすぎ。さらに後述の「返し手勝率」が37.3%まで低下する

### 発見3: ジリ貧仮説は方向的中

新計測「4チェックを返したプレイヤーのその試合の勝率」:

| 構成 | 返し手の勝率 |
|---|---:|
| heavy+(現行) | 55.8% |
| heavy | 50.9% |
| heavy+②(R1) | 46.9% |
| standard | 47.2% |
| standard+dm2+②(R6) | 37.3% |

攻撃行動が安く・多くなるほど「王手を返したのに負ける」が増える。勝敗別使用でも、**全構成で敗者側の再命令使用が勝者側を上回る**(R1: 敗者2.13回 vs 勝者1.33回)——「劣勢側が再命令で粘るが、そのマナ消費で捲りきれない」という仮説のメカニズムと整合する。

**読み方の注意(重要)**: この数字は「返す方が損」という意味では**ない**。比較対象は「返さなかった場合」ではなく(返せなければ多くの場合そのまま5マス目を取られて敗北)、返すことは常に圧倒的に得である。

勝率低下の主因は**返せる層の拡大**にある: 返しが高価な現行では「余裕のあるプレイヤーしか返せない」ため返し手の勝率が高く(55.8%)、返しが安くなると「これまで返せなかった劣勢のプレイヤーまで返せる」ようになり(返答成功1.39→1.69回/試合に増加)、その分だけ平均勝率が薄まる。病院が重症患者も受け入れるようになると生存率の数字が下がるのと同じ構造で、**治療(返し)が弱くなったのではなく、救える範囲が広がった**ことの裏面である。

その上で、ジリ貧仮説(返しに使った霊力の分だけ盤面再建が遅れ、捲りきれない)も敗者側の使用超過(全構成で敗者>勝者)と整合しており、両方の効果が混ざっていると読むのが妥当。分離するには「返答に失敗したプレイヤーの勝率」の計測が必要(次回ランで追加予定)。

### 発見4: ②を入れると単独回転(①)はほぼ絶滅する

①の使用は0.30→0.03/試合。フルコストで回転だけを行う行動は②(同コストで攻撃つき)にほぼ完全に食われる。①は「攻撃対象がいない時の構え直し」としてルール上残す価値はあるが、実質②の付属品になる。想定内の副作用として記録。
