# 属性移行マッピング表 ドラフト(旧: 火水土木無 → 新: 地水火風空)

Status: 提案(レビュー待ち)
Date: 2026-07-05
対象: 検討事項トラッカー #8

## 基本マッピング(機械変換)

| 旧属性 | 新属性 | 根拠 |
|---|---|---|
| 火 | 火 | そのまま |
| 水 | 水 | そのまま |
| 土 | **地** | 名称変更のみ |
| 木 | **風** | 対立ペアの構造を保存(旧: 火↔水・土↔木 → 新: 火↔水・地↔風) |
| 無 | **空** | 中立属性の役割そのまま |

この機械変換は**対立関係とプール内の属性バランスを完全に保存する**ため、バランス面の再調整が不要。フレーバーの不一致(下記)は改名で対応することを推奨。

## 属性別の枚数(変換後)

| | 火 | 水 | 地 | 風 | 空 |
|---|---:|---:|---:|---:|---:|
| 式神(70) | 14 | **19** | 13 | 14 | 10 |
| 霊具(34) | 6 | **10** | 9 | 2 | 7 |

観察: 水がどちらも突出。新盤面の推奨配置は各属性2マス(空1マス)の均等なので、**水カードはプール内で同じ2マスを取り合う**ことになる。プール改訂(フェーズ6)で属性の枚数を均す際の基礎データ。風の霊具が2枚しかない点も要補充。

## 要対応カード(木→風のフレーバー不一致)

木(植物)フレーバーの名前のまま風属性になると違和感が出るカード。**推奨対応: 属性は機械変換どおり風とし、名前を風フレーバーに改名**(属性を地に振り替えるとプールの属性バランスが崩れるため):

| カード | 現名称の問題 | 対応案 |
|---|---|---|
| 大樹の祭主(C5) | 大樹=植物 | 改名(例: 嵐柱の祭主) |
| 大樹の守人(C6) | 同上 | 改名 |
| 苗床の魔導士(C4) | 苗床 | 改名 |
| 芽吹きの種(C1→2) | 種 | 改名 |
| 茨の絡みつき(C4) | 茨 | 改名(例: 旋風の絡め取り) |
| 千の蔓植物(C4) | 蔓植物 | 改名 |
| 木霊の祈祷師(C3) | 木霊 | 改名(風の精霊系へ) |
| 萌芽のスペル(霊具) | 萌芽 | 改名 |
| 大地の塗り替え(霊具) | **地フレーバーなのに旧木属性** | こちらは**属性を地に振り替え**が自然(風の霊具は2枚しかないが、内容が「マスの属性変更」なので地が合う) |

問題なし(そのまま風でよい): 鼓舞の旗印、鼓舞の連鎖、隠れ鎌のリザード、古龍の血脈、駆けの斥候、挑発の槍兵、偏食家(下記参照)

## 効果テキストの属性参照(改訂が必要)

| カード | 現テキスト | 対応 |
|---|---|---|
| 偏食家 / キーワード「偏食」 | 「木属性のクリーチャーまたは木属性フィールドにしか攻撃できない」 | 「風属性の式神または風属性のマス」に書き換え。キーワード定義(keyword_effects)側も同時に |
| 反属性の呪い | 「反属性に変更」 | 新対立(火↔水、地↔風)を参照する定義に。**空属性の式神が対象の場合の裁定が必要**(対立なし→対象にできない、を推奨) |
| 色彩の供犠 | 「捨てたカードの属性に変更」 | 空属性カードを捨てた場合「マスを空にできる」ことになる。許容するか裁定が必要(空マス=誰も修正を受けない、として許容を推奨) |
| 大地の塗り替え | 「指定した属性に変更」 | 空を指定できるかの裁定(同上) |

## 適用手順(フェーズ6)

1. 本表のレビュー・改名案の確定(改名はチームのフレーバー判断)
2. cards.json の attribute を機械変換(スクリプト1本)+ 大地の塗り替えのみ地へ
3. keyword_effects(偏食)と効果テキスト3枚の書き換え
4. スキーマの属性enumを `地/水/火/風/空` に更新
