## **1\. ようこそ**

「異能学園総選挙」へようこそ。

このゲームは、6つの派閥から自分の戦略を選び、3×3の盤面で異能力を駆使して戦う2人用対戦カードゲームです。

### **このゲームが他のTCGと違うところ**

* デッキ構築不要：派閥ドラフトでデッキは自動的に決まる

* 位置取りが重要：3×3の小盤面で、向きと位置で勝負が決まる

* 必殺技のジレンマ：勝利点（VP）を消費して必殺技を発動できる

プレイ時間は 30〜45分。慣れれば20分台で1試合。

## **2\. ゲームの目的・勝利条件**

### **勝利条件**

以下のいずれかを 先に達成した方が勝利 します。

**条件 / 詳細**

* VP勝利: 15VP を先に獲得する

* 領土勝利: 自分のターン終了時に 盤面5マス以上 を占拠している

* タイムアウト勝利: 50ターン経過時、VPが多い方が勝利

### **VPの稼ぎ方**

**行為 / 獲得VP**

* 敵キャラ撃破（コスト1〜2）: 1VP

* 敵キャラ撃破（コスト3〜4）: 2VP

* 敵キャラ撃破（コスト5）: 3VP

### **マナ獲得（自軍キャラ撃破時）**

自分のキャラが撃破された場合（攻撃・反撃問わず）、**撃破されたキャラのオーナーがマナ +1** を得る。

## **4\. ゲームの準備**

### **手順**

17. 派閥ドラフトを行う（次項参照）

18. ドラフトで決まった派閥のキャラ24枚＋アイテム6枚＝30枚デッキを作成

19. デッキをよくシャッフル

20. それぞれ 5枚 ドロー

21. 盤面マットと属性タイルを中央に配置（属性配置はランダム）

22. ジャンケンで先手を決める（先手プレイヤー＝Player 1）

23. ターン1から開始

### **派閥ドラフト（スネーク方式）**

選択順は P1 → P2 → P2 → P1 → P1 → P2 の6ピック。

各プレイヤーは:

* 派閥 2つ（6派閥から）

* アイテムセット 1つ（4セットから）

を選択。同じものは取れない。

### **派閥一覧**

**派閥 / 二つ名 / 色 / 戦略**

* 派閥: B: タンク / 二つ名: 蒼盾 / 色: 青 / 戦略: 防御・カバー・領土勝利

* 派閥: C: コントロール / 二つ名: 紫闇 / 色: 紫 / 戦略: 妨害・操作・洗脳

* 派閥: D: シナジー / 二つ名: 翠光 / 色: 緑 / 戦略: 連携・マーカーバフ

* 派閥: E: スナイプ / 二つ名: 銀眼※ / 色: オレンジ / 戦略: 狙撃・遠距離

* 派閥: F: トリック / 二つ名: 鉛影※ / 色: 灰 / 戦略: 撹乱・潜伏・位置入替

## **5\. ゲームの流れ**

### **1ターンの構造**

①ターン開始 → ②マナ+2／1ドロー → ③メインフェイズ → ④ターン終了

### **①ターン開始**

* 自分の盤面のキャラ全員の「行動済み」「回転済み」フラグをリセット

* 状態異常の継続時間を1ターン進める

### **②マナ獲得・ドロー**

* マナ \+2 （上限なし）

* 山札から 1枚ドロー

### **③メインフェイズ**

以下のいずれかを行う:

**行動 / 内容 / コスト / ターン継続**

* 行動: 召喚 / 内容: 手札から1枚召喚 / コスト: 召喚コスト / ターン継続: 召喚=ターン即終了

* 行動: 再行動 / 内容: 既存味方を1体動かす / コスト: 再行動コスト / ターン継続: 継続（複数キャラを順次行動可）

* 行動: アイテム使用 / 内容: 手札のアイテムを発動 / コスト: アイテムコスト / ターン継続: 継続

* 行動: ウルト発動 / 内容: エースキャラの必殺技 / コスト: VP / ターン継続: 継続（行動消費）

* 行動: パス／ターン終了 / 内容: 何もしない / コスト: なし / ターン継続: 終了

### **④ターン終了**

* VP計算

* 勝利条件チェック

* 領土勝利チェック（自分のターン終了時のみ）

* ターンプレイヤーを切り替え

## **6\. 召喚**

### **基本**

24. 手札からキャラカード1枚を選ぶ

25. 召喚コスト分のマナを支払う

26. 盤面の 空いているマス に向きを選んで配置

27. 召喚時効果が発動

28. 自動攻撃: 攻撃範囲に敵がいれば自動的に1回攻撃

29. このターンは即終了

### **配置ルール**

* 初回の召喚（盤面に味方0体）: どのマスでも可

* 2回目以降: 既存の味方または敵の 隣接マス（縦・横）にのみ召喚可能

* 盤面上限: なし

### **コスト軽減（エース回収条件）**

コスト5のキャラ（エース）は、各派閥の特定条件を満たすたびに コスト-1（最低2）。

**派閥 / 条件**

* アグロ: このターン攻撃した味方の数

* タンク: カバー持ちで隣接味方がいるキャラの数

* コントロール: デバフ状態の敵の数

* シナジー: マーカー持ちの味方の数

* スナイプ: 照準が付いた敵の数

* トリック: 敵のB位置にいる味方の数

## **7\. 再行動（攻撃・回転）**

すでに盤面にいる味方は、マナを支払って 再行動 できます。

### **再行動の種類**

**行動 / 内容**

* 攻撃: 攻撃範囲の敵を攻撃（戦闘ルール参照）

* 回転: キャラの向きを変える

※ 「移動」「位置入替」は通常の再行動ではなく、特定カードの効果（押し出し等）でのみ発生する。

### **再行動コスト**

各キャラに 再行動コスト が設定されている（カード上 ACT）。

* 通常コスト: 1〜3

* action\_tax（再行動コスト+1）を受けていると追加コスト

* reduceactivatecost で軽減可能

### **制限**

* 同じキャラは 同ターン中、攻撃は1回まで

* 回転 は別カウントで、攻撃とは独立に1回まで

* ターン終了までに何体動かしてもOK（マナがあれば）

## **8\. 戦闘**

### **攻撃の手順**

30. 攻撃するキャラを選ぶ

31. 再行動コスト分のマナを支払う

32. 攻撃範囲 の中から対象を1体選ぶ

33. 反撃可能性をチェック → ダメージ計算 → ダメージ適用

34. HP0以下のキャラは撃破→VP獲得

### **攻撃範囲**

カード下部の 攻撃範囲グリッド で表示。中央★が自キャラ、塗りマスが攻撃対象。

主な攻撃範囲:

**範囲名 / 形 / 説明**

* 範囲名: 正面1 / 形: 1マス前方 / 説明: 最も基本

* 範囲名: 前後 / 形: 前1＋後1 / 説明: 背後にも届く

* 範囲名: 直線2 / 形: 前2マス / 説明: 槍系

* 範囲名: 前列 / 形: 正面の行3マス / 説明: 横一閃

* 範囲名: 十字 / 形: 上下左右1マス / 説明: 4方向同時

* 範囲名: L字（左/右） / 形: 前1＋横1 / 説明: 同時攻撃

* 範囲名: 狙撃 / 形: 5マス先まで / 説明: 遠距離

* 範囲名: 全域 / 形: 全マス / 説明: 魔法

### **B位置（ブラインドスポット）**

敵の死角から攻撃 すると以下のボーナス:

* ATK \+1（**物理攻撃のみ**。魔法攻撃ではB位置ボーナスは発生しない）

* 反撃を受けない

死角は通常 キャラの背後 だが、blindPattern により異なる:

**パターン / 死角**

* back: 背後のみ

* sides: 左右のみ

* back\_sides: 背後＋左右

* all: 全方向（隣接全部）

* none: 死角なし（全方向反撃）

### **ダメージ計算**

ダメージ \= ATK    \+ B位置補正 (+1)    \+ 各種バフ    \- 防護 (-1)    \- チームDR

* 防護：ダメージ-1（最低0）。貫通 で無視。

* 回避：物理攻撃を1回無効化。貫通 で無視。魔法 には無効。

* 反射：魔法ダメージを攻撃元に返す

* damage\_link：受けたダメージの半分を隣接味方に分散

### **反撃**

* 攻撃を受けたキャラは反撃を試みる（**物理攻撃のみ**。魔法攻撃では反撃しない）

* 反撃側の 正面が攻撃元の方向 にある場合のみ反撃成立

* B位置から攻撃された場合、反撃不能

* 要塞 キーワード持ちは自分から攻撃できないが、反撃は可能

* 反撃は反撃を受けない（再反撃は発生しない）

* 反撃はカバーされない（カバー役は割り込まない）

* 反撃で受けたダメージは「on\_damaged効果」を発動させない

### **カバー時の反撃ルール**

* カバーが発動した場合、原則として反撃は発生しない（攻撃元・カバー役・元のターゲットのいずれも反撃しない）

* 例外: カバー役が 【要塞】 を持つ場合、カバー役自身が反撃する（先制反撃含む）

\- 不動のナギサ（要塞＋カバー）が該当

### **先制（quickness）**

* 反撃側が 先制 を持っていれば、攻撃される前に反撃を実行

* B位置から攻撃された場合は先制も無効

### **撃破とVP**

* HP0以下になったキャラは即時撃破

* 撃破した側に コスト相応のVP が入る

\- コスト1〜2 → 1VP

\- コスト3〜4 → 2VP

\- コスト5 → 3VP

## **9\. 属性とマス**

### **5つの属性**

**属性 / 漢字 / 読み / 色**

* 属性: faust / 漢字: 拳 / 読み: ファウスト / 色: 朱

* 属性: geist / 漢字: 念 / 読み: ガイスト / 色: 紫

* 属性: licht / 漢字: 光 / 読み: リヒト / 色: 金

* 属性: nacht / 漢字: 闇 / 読み: ナハト / 色: 紺黒

* 属性: nicht / 漢字: 虚 / 読み: ニヒト / 色: 銀白

### **対立構図**

拳 (faust) ⇔ 念 (geist)光 (licht) ⇔ 闇 (nacht)虚 (nicht) は中立（誰とも対立しない）

### **属性マスの効果**

**状況 / 効果**

* 自分と同じ属性のマスに乗る: HP \+2（召喚時のみ）

* 対立属性のマスに乗る: HP \-2（召喚時のみ）

* 中立（虚）または不一致: 影響なし

虚（nicht）は中立: 自分が虚でも、マスが虚でも、HPボーナス・ペナルティはなし。

### **盤面の属性配置**

3×3盤面の9マスに属性が割り当てられる。配置はランダムまたは固定（オプション）。

標準配置例:

* 拳×2 / 念×2 / 光×2 / 闇×2 / 虚×1

## **10\. キーワード一覧**

### **防御系**

**キーワード / 効果**

* 防護: 受けるダメージを-1（最低0）。消費しない

* 回避: 物理攻撃を1回無効化。マーカー として消費される

* 要塞: 自ら攻撃できない。反撃のみ可能

* 不動: 押し出し・引き寄せの効果で動かない

* damage\_link（分散）: 受けたダメージの半分を隣接味方に分散

* 反射: 受けた魔法ダメージを攻撃元に返す

### **攻撃系**

**キーワード / 効果**

* 貫通: 攻撃時、防護・回避を無視

* 先制: 反撃時、攻撃側より先にダメージを与える

* stealth（隠密）: 敵から効果対象に取れない。攻撃が常にB位置扱い

* pressure（圧力）: 隣接する敵の再行動コストが+1

### **補助系**

**キーワード / 効果**

* カバー: 隣接味方が受けるダメージを代わりに受ける（防護・回避は元のターゲット基準で計算）

* 召喚制限: 盤面に味方4体以上いる時のみ召喚可能

* 照準（マーカー）: 効果対象を識別するマーカー。スナイプ派閥が参照

### **マーカーバフ（消費型）**

特定の効果（カードや【カバー】等）でキャラにマーカーとして付与される。

対応するキーワードと同じ効果を発揮する。1回消費したら消える。

**マーカー / 同等のキーワード / 効果**

* マーカー: 防護マーカー / 同等のキーワード: 防護 / 効果: 受けるダメージ-1

* マーカー: 回避マーカー / 同等のキーワード: 回避 / 効果: 物理攻撃を1回無効化

* マーカー: 貫通マーカー / 同等のキーワード: 貫通 / 効果: 攻撃時、防護・回避を無視

* マーカー: 先制マーカー / 同等のキーワード: 先制 / 効果: 反撃時、先に攻撃

重要: ルール上「防護を持つ」と書かれた箇所は、キーワードでもマーカーでも適用される。例えばカバー役が防護マーカーを持っていれば、カバー時のダメージから-1される。

## **11\. ウルティメット（VP消費必殺技）**

### **概要**

12体のエースキャラ（各派閥2体）には固有の必殺技 \= ウルト がある。

### **共通ルール**

**ルール / 詳細**

* 使用回数: 1ゲームに 1回のみ（同じカードIDで）

* コスト: VP（勝利点）2〜3を消費

* 発動後: そのキャラはそのターンに 再行動できない（攻撃も回転も不可）

* 召喚ターン制限: 一部のウルトは召喚した次のターンから使用可能（カードに記載）

注意: ウルトを使ったキャラだけ行動不可になる。他の味方は通常通り再行動できる。

### **ウルト一覧**

**派閥 / カード / ウルト名 / 効果 / VP / 召喚ターン**

* 派閥: Aggro / カード: 穿陣のセラ / ウルト名: 穿陣突破 / 効果: 次の攻撃に貫通＋射程・障害無視 / VP: 2 / 召喚ターン: 次ターン以降

* 派閥: Tank / カード: 誓盾のルミナ / ウルト名: 絶対盾 / 効果: 次の敵ターン中、味方全員の被ダメ-1 / VP: 2 / 召喚ターン: 召喚ターン可

* 派閥: Tank / カード: 不動のナギサ / ウルト名: 不動の聖域 / 効果: 隣接味方を全回復＋防護 / VP: 3 / 召喚ターン: 召喚ターン可

* 派閥: Control / カード: 簒奪のオルド / ウルト名: 精神簒奪 / 効果: 敵1体を3ターン洗脳 / VP: 3 / 召喚ターン: 次ターン以降

## **16\. 困ったときは**

### **よくある混乱ポイント**

### **「召喚」と「アイテム使用」の違い**

* 召喚 \= キャラを盤面に出す → ターン即終了

* アイテム使用 \= カード効果発動 → ターン継続

### **再行動と回転の違い**

* 攻撃 \= 「行動」フラグを使う（同ターンで1回まで）

* 回転 \= 「回転」フラグ（独立、同ターンで1回まで）

* 別カウントなので、同じキャラが「攻撃して、その後回転する」は可能

### **B位置の判定**

* 攻撃側の位置が 守備側の死角の方向 にある場合のみB位置成立

* 守備側の blindPattern を確認

### **マーカーの消費**

* マーカーバフ（has\_protection など）は 使ったら消える

* キーワード持ちのバフ（カードの keywords に書いてある）は 永続
