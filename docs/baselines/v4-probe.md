## 3. 考察

既存の確定KPI帯との比較では 1/12 項目が帯内だった。先手勝率は2000試合では約±2.2ptの幅があるため参考判定とし、それ以外の帯外項目を切り分け対象とする。なお、この帯は純粋素体で確定した基準であり、形状環境では改訂余地がある。ただし、対照の印刷仕様からの変化も大きいため、今回は単なる帯の不一致ではなくゲーム構造が実際に動いたと読む。

全体として、ゲーム速度は対照から遅くなり、撃破量は増加した。平均盤面数は4.77→4.42へ減り、占拠勝ちは99.2%→77.1%、生命勝ちは0.7%→21.6%へ移った。つまり「5体を並べ切る競争」より、式神を繰り返し処理して生命を削る試合が増えた。

撃破数は4.49→11.34、撃破時獲得霊力は4.49→22.68、攻撃系再命令は3.64→8.92へ同時に増えた。これは実測上、撃破・霊力回収・追加行動が循環する環境になったことを示す。ただし、撃破時霊力2だけでなくHP低下、陰陽、太極、弱点補正も同時に変えているため、どの変更が循環を開始・増幅したかはこのランだけでは分離できない。

弱点撃破率は50.4%→24.8%へ半減した。総撃破が増えている一方で位置取り由来の撃破比率が落ちているため、今回の合成環境では「弱点を取ること」より通常範囲での反復攻撃が撃破の中心になった。チェック返し率64.2%も帯上限を超えており、王手は返しやすくなったが、その代償として試合が長期化し、生命決着が増えている。

先手勝率差は ±2.2ptを超えているが、この試合数だけで確定判定はせず、必要なら8000試合以上で追試する。

太極は全ターンの75.3%で占有され、軽減は先手2.72回/試合、後手1.98回/試合発動した。先手は占有でも41.8%対33.6%、軽減でも約0.74回/試合多い。同時に先手勝率が57.3%まで上がっているため、太極の先後利用差は最優先の疑義である。ただし相関段階なので、太極だけを外したD1で確認するまで原因とは断定しない。

陰陽修正は+2が7.85回/試合、-2が0.46回/試合で、AIは不利属性をほぼ避けていた。属性システムは主に耐久上昇として作用しており、対立属性のリスクは現状では弱い。5体目が召喚反撃で崩れたケースは0件で、この例外処理は今回の決着差を説明しない。

## 4. 切り分け候補（優先順）

1. **太極軽減のみOFF**: 先手の占有・軽減差と57.3%の先手勝率が同時に出たため最優先。陰陽盤面は残し、太極の経済効果だけ外して先後差を識別する。
2. **撃破時霊力を1へ戻す**: 22.68霊力/試合と8.92再命令/試合の循環を直接検査する。太極OFFとは別ランにして寄与を混ぜない。
3. **C2/C3 HPを印刷仕様へ戻す**: 低コスト撃破ラインが総撃破11.34にどれだけ寄与したかを単独で測る。
4. **弱点ボーナスを+2へ戻す**: 弱点撃破率24.8%の回復を確認する。総撃破や試合長への影響とは分けて読む。
5. **陰陽盤面を旧4属性盤面へ戻す**: 太極を切り分けた後、陰陽+2/-2そのものの耐久・配置影響を測る。

この順序は寄与の推定ではなく、変更範囲の大きさと次ランでの識別性に基づく。

## 5. 実装ノート

- 7/18印刷仕様の9種16枚、攻撃/反撃形状、heavy再命令+回転攻撃、先手3/後手4を維持。
- v4だけHPをC2=2/C3=3/C4=5、撃破時霊力2、弱点物理ボーナス+1へ差し替えた。
- 各デッキは陰8/陽8の袋をコストと独立にシード下でシャッフル。盤面は `空陰空/陰太極陽/空陽空` 固定。
- 太極は中央を所有する側の召喚支払いだけ-1（下限1）。再命令には適用しない。軽減後コストを召喚可能判定・オラクル・実支払いへ共通適用した。
- 陰陽+2/-2の発動数は、移動のない素体環境なので召喚時に修正が適用された回数として計測した。
- 勝利条件、pending check、depth-6返答探索、AIスコア重みは変更していない。
- 配置ロジックも比較可能性を優先して既存実装を変更していない。現行シミュ実装は自軍式神に上下左右隣接する空きマスを返すため、指示書の『敵味方問わず』というルール文言とは差がある。実ゲーム正本へ合わせる場合は、別タスクで対照群から再計測が必要。
- 5体目崩壊は、召喚前4体かつ召喚した式神自身が召喚攻撃への反撃で撃破されたケースのみを数えた。
