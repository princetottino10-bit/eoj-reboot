# 異能学園総選挙 テストキット ルールブック

4陣営テストキット（顕現CIP / 猛攻aggro / 守護defense / 詠唱spell）で確定しているルールをまとめる。
ベースは PS3「The Eye of Judgment」の3×3制圧メカニクス。詳細な原作ルールは
`異能学園総選挙_dev_briefing.md` を参照。

---

## 1. 盤面

- **3×3グリッド＝9マス（9選挙区）** の共有盤面。
- 各マスには固定属性を割り当てる（テストキット既定配置）：

  ```
  水 火 木
  土 無 水
  火 木 土
  ```

  ※ 各属性2マス＋中央=無（中立）1マス。プレイマットの各マス右上に属性を表示。

---

## 2. 勝敗条件 〔確定〕

- **勝利**: 次のいずれかを満たした側が勝利。
  1. ターン終了時に **9マス中5マスを支配**（過半数＝選挙勝利）。
  2. **VPを15点獲得**。
- **4マス支配＝チェック**（あと1マスで敗北）。
- **デッキ切れによる敗北は無し** 〔確定〕。山札が尽きたら切り直して継続する。

---

## 3. VP（ビクトリーポイント）

- クリーチャー撃破で獲得。獲得点はそのクリーチャーのコストに依存：
  - コスト **2以下 → 1点**
  - コスト **3〜4 → 2点**
  - コスト **5以上 → 3点**
- **自滅（自軍効果や巻き込みでの撃破）でもVP・マナは入る** 〔確定〕。
- マス制圧でもVPを獲得する（カード／効果に準じる）。
- 15点で即勝利。

---

## 4. ターン構造

1. **ドロー**: カード1枚＋マナ2を獲得（先攻1ターン目はドローなし）。
2. **アクション**（マナの続く限り、任意の順・複数回）:
   - スペルの使用
   - 既存クリーチャーの攻撃（アクティベーションコスト消費）
   - 既存クリーチャーの回転90°（各クリーチャー1ターン1回）
3. **召喚**: クリーチャー1体を配置 → **ターン強制終了**。
   - 召喚はターン最後のアクション。アクションは召喚より前に行う。
   - 初回は任意マス、以降は既存クリーチャー（敵味方問わず）に隣接するマスのみ。
   - 召喚時に向き（上下左右）を選択。
   - **召喚ロック**: 盤面にクリーチャーが4体以上いないと召喚できないカードがある。
4. **リゾリューション**: 手札7枚チェック（超過は捨て札）→ 勝利判定。

---

## 5. マナ

- 毎ターン開始時に+2。
- クリーチャーが撃破されると、そのオーナーに+1。
- 上限なし（蓄積可能）。
- 用途: 召喚 / 攻撃（アクティベーション）/ 回転 / スペル使用。

---

## 6. 属性と地形

- **5属性**: 火 / 水 / 土 / 木 / 無。
- **対立関係**: 火 ⇔ 水、土 ⇔ 木。無は中立。
- **地形ボーナス**: 同属性マスに召喚 → **HP +2**。
- **地形ペナルティ**: 対立属性マスに召喚 → **HP −2**（HP0以下で即死）。
- 無属性クリーチャー／無マスは変動なし。

---

## 7. 攻撃と戦闘

- 各クリーチャーは固有の攻撃パターンを持つ。座標は **[前方, 左右]**、中央=自機（上向き基準、向きで回転）。
- **回避は無し**。攻撃範囲内にいれば必ず当たる＝向きと位置取りがそのまま「回避」として機能する。
- **同士討ち（巻き込み）** 〔確定〕:
  - 単体攻撃では味方を狙えない。
  - **複数同時攻撃で攻撃範囲に味方が入っている場合、味方にも攻撃が当たる。**
- **反撃**: 攻撃を受けた方向が自分の攻撃範囲内なら自動で反撃。
- **弱点（死角）**: 死角方向からの攻撃はダメージ+1・反撃不可。
- **物理攻撃 / 魔道攻撃**: 魔道攻撃は盤面全域を対象に選べる（位置無視）。攻撃補正を行わず、反撃を受けない。

---

## 8. ウルト（デッキ外カード）

- 各陣営に2枚のウルトが存在。**デッキビルド時に1枚を選択**してデッキ外に置く。
- **共通解禁条件**: フィールドのクリーチャー合計が6体以上・1ゲーム1回。
- 解禁後は自分のターンの任意タイミングで発動。**陣営固有コスト**を支払う。

---

## 9. デッキ構成

- デッキ30枚（クリーチャー＋スペル）。同名カード最大3枚（一部1枚制限）。
- 初期手札5枚。マリガン1回可（全戻し・引き直し）。
- 手札上限7枚。
- ウルトはデッキ外（上記8）。

---

## 10. キーワード

| キーワード | 効果 |
|------------|------|
| 連撃 | 再行動コスト1回分で2回攻撃できる |
| 挑発 | 召喚時およびターン開始時、隣接する敵クリーチャー全ての向きをこのクリーチャーの方向に強制変更する |
| 先制 | 防御時、攻撃を仕掛けた相手より先に反撃する |
| 不屈 | 撃破されるとき、ゲーム中1回まで、破壊される代わりに体力1で残る |
| 暗殺 | 対象の弱点範囲を突いて攻撃したとき、さらに追加で1ダメージを与える |
| 重ね召喚 | 場の味方クリーチャー1体を生贄に捧げ、そのマスに重ねて召喚できる。マナコストは差額分のみ。召喚されたターンは攻撃不可、ターン終了とならない |
| 魔道攻撃 | フィールド全体から対象を選べる。攻撃補正を行わず、反撃を受けない |
