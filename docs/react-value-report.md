# 再命令の損得レポート(R1構成・対称fork因果比較)

Date: 2026-07-10
生データ: [baselines/react-value-forked-v2.md](baselines/react-value-forked-v2.md) / react-value-forked-v2-results.json
Status: 結果報告(採用済みR1構成に対する事後検証)

---

## 0. 何を知りたかったか

採用済みのR1構成(再命令コスト heavy 2/3/3/3/3/4 + 回転攻撃ルール)で、**「再命令は"使うのが正解"になっているか」**を因果的に確かめる。

- 正解なら → コスト設計は完成。フェーズ4で再命令に価値を足す(割引霊具等)必要はない
- 使わない方が勝つ局面ばかりなら → コストは触らない方針なので、フェーズ4で再命令側に価値を足す設計が要件になる

## 1. 方法(と、最初の測定を捨てた話)

**因果比較(fork法)**: 同一局面からゲームを2つに分岐し、片方は再命令を使い、もう片方は使わずに、両方を同じ乱数で最後まで走らせて勝率差(Δ)を測る。同一局面から分岐するので「劣勢な人ほど使う」という相関の罠(傘の誤謬)が原理的に消える。

### 最初の測定は設計ミスで捨てた(教訓)

初回の測定は Δ−20.7pt(=「再命令は大損」)という極端な値を出したが、**これは実験のバグだった**。原因は2つ:

1. **比較が非対称**: 「使う世界」は手を強制したのに、「使わない世界」はAIが残りから選び直せた。選び直せる方が有利なのは当たり前で、その証拠にミラー戦なのに「使わない世界」の勝率が53.8%(=50%であるべき)だった。
2. **使わない局面まで強制**: AIが本来やらない局面でも無理に使わせていたので、「やりたくない手を強制すれば損」という当たり前を測っていた。

**教訓**: カウンターファクチュアル比較は、両世界の介入を対称にしないと因果を測れない。数字が極端なときは結論より先に設計を疑う。修正版(v2)では「AIが実際に使った局面だけ」を対象にし、両世界とも「1手だけ差し替え、以降は同じAI」に揃えた。**対称性の自己検証(使わない世界の勝率が50%か)をゲートにし、PASS(50.1%)を確認してから結果を読んでいる。**

## 2. 結果

3000試合・分岐3524件。対称性ゲートPASS。

| 種別 | n | Δ(使う−使わない) | 読み |
|---|---:|---:|---|
| 攻撃系ぜんぶ(②+③) | 3524 | **−0.1pt** | AIの判断は損得中立=最適化されている |
| ②回転攻撃のみ | 670 | **+7.0pt** | 回転攻撃には固有の価値がある |
| ③素の攻撃のみ | 2854 | −1.8pt | 単体では損得中立の限界的選択肢 |

## 3. 主要な発見

### 発見1: AIの再命令判断はほぼ最適(Δ≒0)

②+③合算でΔ−0.1pt。「AIが再命令を使うと決めた局面で、実際に使うのと次善手に替えるのが五分」= **使いすぎでも使わなさすぎでもない**。**R1の再命令コストは、使う判断がされる場面において適正価格**と言える。

### 発見2: 回転攻撃ルール(②)には明確な固有価値(+7.0pt)

これが最大の収穫。AIが回転攻撃を選んだ局面で、回転なしの代替に替えると7pt負ける。局面を絞るとさらに鮮明:

| 層 | ②を使う | 代替に替える | Δ |
|---|---:|---:|---:|
| 互角 × 通常 | 68.8% | 36.3% | **+32.5pt** |
| 劣勢 × 通常 | 33.6% | 14.1% | +19.5pt |

回転攻撃を潰された局面の7〜9割は「回転だけ」に落ちる。つまり**「構え(回転)と火力(攻撃)を1手にまとめられること」自体に価値**があり、分離すると効果が半減する。回転攻撃ルールを採用した判断は、因果比較で事後的に正当化された。

### 発見3: 素の攻撃(③)は損得中立

Δ−1.8pt。大きな層を均すとほぼゼロで、「使っても使わなくても大差ない」限界的選択肢。これは以前の観測(平時は攻撃再命令の未使用率が7割)と整合する性格。害はないが、単体では戦略の主役ではない。

### 発見4: 唯一負ける「劣勢×王手被弾中」は敗勢のノイズ(−9.6pt)

はっきり負ける層はここだけ(使う1.9% / 代替11.5%)。ただし**どちらの勝率も一桁%**で、これは「もう詰んでいる盤面では何をしても負ける」局面。再命令が悪いのではなく盤面が既に敗勢。以前見えた「ジリ貧」の残り香だが、環境全体を汚す規模ではない。

## 4. デザインへの含意

1. **再命令コスト(heavy)は確定でよい。** 使う判断がされる場面で損得中立=適正。フェーズ4で「再命令に価値を足す割引霊具」を急いで作る必要はない(当初の懸念は"不要寄り"で決着)。
2. **回転攻撃ルールは"あってよかった"が数字で裏付けられた。** R1採用の目玉として自信を持ってよい。
3. **③単体の凡庸さは伸びしろ。** 素の攻撃をもっと戦略的にしたいなら、フェーズ4で「攻撃するたび〜」系の効果が効く余地がある。必須ではなく味付けの選択肢。
