## 共通ルール(全カード)

- **攻撃範囲=反撃範囲**(自分が攻撃できるマスから殴られたら反撃できる)
- 攻撃は範囲内の**1体を選択**
- 攻撃タイプは**物理**(死角ボーナス+2、反撃あり)
- 弱点(死角)から物理攻撃を受けると**ダメージ+2**・**反撃できない**

図の見方(自分は中央、上が向いている方向):

```
 ┌───┬───┬───┐
 │   │ ● │   │   ● = 攻撃範囲=反撃範囲
 ├───┼───┼───┤
 │   │ 自 │   │   自 = 自分
 ├───┼───┼───┤
 │   │ ✖ │   │   ✖ = 弱点(死角)
 └───┴───┴───┘
```

## 印刷時の留意点

- **ミラー戦なので同一内容を2セット**(計32枚)
- 属性は各プレイヤー内で地・水・火・風を4枚ずつ。カードに書き込むか、別途マーカーで管理
- 盤面の属性配置(固定)も別途用意:
  ```
  水 火 風
  地 空 水
  火 風 地
  ```
- 生命15・霊力のカウンターも必要(初期: 先手3 / 後手4)

## 注意

- **HPのC2=3 / C3=4 は7/18の暫定調整**(素体基準v3.1はC2=2 / C3=3)。形状導入に伴う乱戦を抑えるための現場調整
- 九尾の形はシミュ後に差し替えたもの。確定前に再シミュ推奨
