# デッキ構築ルール ドラフト v0

Status: 提案(未採用・未シミュ)
Date: 2026-07-05

バニラ基準v1.1の体験目標(平均10〜12R、王手応酬、位置取りゲーム)から逆算したデッキ構築ルールの叩き台。効果レイヤーのシミュレーションに入る前に確定させる。

## 前提となる観測

- 現行シミュのデッキは「クリーチャー12枚のみ・単一派閥・全種+重複2枚」の仮組みで、デッキルールは未定義だった。
- 1ターン1召喚+手札5枚補充のため、カード消費は約1〜1.5枚/ラウンド(アイテム実装後)。
- 12枚デッキは8〜10ラウンドで空になり、長期戦(P90帯)は「補充なしの消耗戦」になっていた。タイムアウトの主因。
- 派閥プール: 対象4派閥は各クリーチャー10種。アイテムは派閥ごとに4〜5種。

## 提案ルール

| 項目 | 案 | 根拠 |
|---|---|---|
| 総枚数 | **24枚** | 消費1.5枚/Rで13R前後までデッキが持つ。通常試合(9〜12R)ではデッキ切れせず、長期戦にだけ終了圧がかかる位置 |
| 内訳 | **クリーチャー16 + アイテム8**(シミュ標準構成) | 召喚は約8回/試合なのでクリーチャー16で常に選択肢が残る。対人ルールとしては「クリーチャー14枚以上」の下限制約だけにして比率を自由化する余地あり |
| 同名制限 | **2枚まで** | プール10種で16枠を埋めるには最低8種の採用が必要になり、派閥の顔ぶれが自然に揃う。突出カードのスパムも上限2で抑制 |
| 派閥 | **単一派閥(当面)** | バランス計測の連続性(相性行列)と派閥アイデンティティの確立が先。2派閥混合は効果レイヤー安定後の拡張候補 |
| マリガン | **なし** | 16/24がクリーチャーなら初手にクリーチャー0枚の確率は約0.13%で、救済ルールのコストに見合わない |
| 派閥ウルト | **デッキ外**(1ゲーム1回、発動条件つき、現行どおり) | デッキ圧迫なしの「必殺技」としての現行設計を維持。要確認事項として明記 |

### デッキ切れ処理(ファティーグ)

**ターン終了時の手札補充でデッキが空だった場合、そのプレイヤーはライフを1失う(1ターンに1回)。**

- 狙い: P90帯の膠着(材料切れの消耗戦)に有限の終了圧をかける。タイムアウト(引き分け)をほぼ根絶する。
- デッキ切れは概ね13R以降に始まるので、通常試合(9〜12R)には一切影響しない。
- 既存のライフ資源を使うため、新しい概念の追加コストが最小。
- 敗北の記録上は「ライフ勝ち」に含まれるため、ライフ勝ち率のKPI帯(8%以下)との整合を再計測で確認する。

### 採用しなかった代替案

- **総枚数20**: デッキ切れが10〜11Rに来てしまい、通常試合の体験に干渉する。
- **総枚数30**: 単一派閥・同名2枚だと「全種×2」の強制構成になりデッキ構築の選択が消える。ファティーグも20R以降まで来ず、膠着対策にならない。
- **デッキ切れ=即敗北**: 崖が急すぎて、終盤が「相手のデッキ枚数を数えるゲーム」に変質する恐れ。ライフ1点ずつの方が攻防の余地が残る。

## シミュレーション実装メモ

1. `buildDeck` を変更: クリーチャー16枚(全10種+重複6枚をランダム選択)。当面は重複選択をランダムにする(デッキ構築AIの賢さを計測に混ぜないため)。
2. アイテム8枚は効果レイヤー実装までは**デッキに入れない**(バニラ再計測はクリーチャー16枚デッキで行う)。ダミーカード(何もしないスペル)を入れると手札を永久に圧迫して非現実的な劣化になるため。
3. ファティーグを `drawToFive` / `finishTurn` に実装。
4. 計測追加: デッキ切れ発生ラウンドの分布、ファティーグ由来の敗北率、タイムアウト率の変化。

## 再計測プラン

1. **v1.1 + 新デッキルール(16枚+ファティーグ)** を2000試合、seed `20260702` で回し、v1.1のKPI表と比較する。
   - 予想: 通常帯(中央値9.5R)はほぼ不変、P90とタイムアウトが改善、ライフ勝ち率が微増。
   - ライフ勝ち率が8%を大きく超える場合はファティーグを緩和(2ターンに1回など)。
2. ファティーグon/offのA/Bで、P90への寄与を分離確認する。
3. 問題なければこれを **バニラ基準v1.2(デッキルール確定版)** として凍結し、効果レイヤー(アイテム8枚を実スペルにする)はこの上で実装する。
