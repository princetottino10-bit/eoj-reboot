## 2. 効果設計ガイドライン違反の候補(リスク監査)

### 🔴 高リスク: 無限ループの部品になりうる(フェーズ4で最優先確認)

| カード | 問題 | ガイドライン |
|---|---|---|
| **巡る輪廻**(霊具・コスト0) | 墓地の式神1体をデッキ下に戻して**霊力+3**。0コストで純増+3は原則1に真っ向から違反。墓地リシャッフル環境では弾も無限 | 原則1 |
| **詠唱の極致**(式神C7) | 自分の霊具使用のたび「ダメージ+1・**霊力+1・1ドロー**」で回数無制限。0〜1コスト霊具が実質無料+ドロー付きになり、霊具連打が自走する | 原則1・2・5 |
| **残響の詠唱者**(式神) | 自分の霊具が墓地に行く代わりにデッキ下へ戻る=**霊具が消費されない**。経済系霊具と組むと循環の骨格になる | 原則1 |
| **鋳潰しの炉/払い戻し**(霊具・コスト0) | 手札を霊力に変換。単体は等価交換だが、ドローエンジンと組むと「手札→霊力」の無限変換弁になる | 原則1 |
| **万象の鼓動**(霊具・コスト0) | 自式神を破壊して「コスト+2」の霊力。**dm裁定で+1がさらに乗り、実質コスト+3**。テキストの数値見直しが必要 | 原則1+dm裁定 |
| **詠唱の連鎖**(霊具・コスト0) | 霊具1枚を2回発動+1ドロー。倍化装置。詠唱の極致・残響と同席させてはいけない | 原則3 |

### 🟡 中リスク: 回数制限のない「〜するたび」トリガー(原則5違反)

- **風見の哨戒兵**: 移動・向き変更のたび1ドロー(無制限)——回転がコスト1級の行動である以上、ドローエンジン化する
- **道標の渡り鳥**: 移動・向き変更のたびATK+1(そのターン)——回転連打と組むと1ターンで大幅増
- **詠唱の弟子**: 霊具使用のたび1ドロー(無制限)
- **占術の対価**: 召喚時効果の発動のたび霊力+1(無制限)——コピー系と直結

対処はテンプレ「1ターンに1回まで」の付与で足りる見込み(原則5)。

## 4. 個別偏差表の初期候補(カーブから意図的に外すカード)

カーブ機械適用で**カードの固有性が壊れる**もの。偏差表(カーブ外の例外リスト)に載せる候補:

| カード | 問題 | 偏差案 |
|---|---|---|
| 鋼の継承者 | 設計上ATK0(墓地から継承して育つ)がカーブでATK2になる | ATK0を維持し、その分HP or 効果を強化 |
| 監視者の塔 | 攻撃不可(attack_mode: none)なのでカーブのATKが無意味 | ATK0固定。コストは効果(メタ監視)基準で値付け |
| 偏食家 | 全域術式だが対象制限(風属性のみ)つき。無条件の全域と同じ税は過剰 | 術式税ATK-1を免除 or コスト-1(フェーズ3で制限付き範囲の価格を測ってから) |
| 誘引のセイレーン/沈黙の祠 | 全周反撃・弱点なしという防御形状はカーブ+現行税で表現できない | フェーズ3に「反撃範囲・弱点の形状価格」の測定を追加(下記) |

**フェーズ3への申し送り**: 現行の形状税は攻撃側(範囲・対象数)しか課税していないが、実プールには**反撃範囲・弱点配置のバリエーション**(全周反撃、弱点なし、側面弱点)が多数ある。防御形状の価格測定をフェーズ3の実験に含めるべき。
