# Card Text Style Guide

カード効果文の表記ゆれを減らすための暫定ルールです。効果の意味を変えずに機械的に寄せられるものと、ルール判断が必要なものを分けています。

## 基本表記

- プレイヤー視点は `自分` / `相手` を使う。
- 所有を示す場合は `自分のクリーチャー` / `相手のクリーチャー` を使う。
- `自軍` は原則 `自分の` に置き換える。
- `敵クリーチャー` は原則 `相手のクリーチャー` に置き換える。
- `カードを1枚引く` を使い、本文では `ドロー` を使わない。
- `マナを1得る` を使い、本文では `マナ+1` を使わない。
- 本文では `体力` / `攻撃力` を使い、カードステータス欄以外では `HP` / `ATK` を避ける。
- `コストを支払わず` を使い、本文では `コスト無料` を避ける。
- `空いているマス` を使い、本文では `空きマス` を避ける。
- `デッキの上からN枚` を使い、本文では `デッキトップ` を避ける。
- 永続修正は `永続的に攻撃力+1する` のように本文へ組み込み、`（永続）` だけにしない。

## タイミング表記

- ターン中の効果は `このターン中` を使う。
- 相手ターンを指す場合は `次の相手のターン中` を使う。
- 継続期限は `次のターン終了時まで` のように `まで` を付ける。
- 発生タイミングは `ターン終了時、` のように読点を付ける。
- 回数制限は `1ターンに1回まで` の形に統一する。

## 用語の使い分け

- `撃破` は戦闘やダメージで場から離れることに使う。
- `破壊` はカード効果で明示的に場から離すことに使う。
- `除外` は墓地にも手札にも戻らない領域へ置く場合のみ使う。
- `召喚時、` はクリーチャー自身の効果に使う。
- スペルが対象を召喚する場合は `召喚されたクリーチャーは...` のように主語を明示する。
