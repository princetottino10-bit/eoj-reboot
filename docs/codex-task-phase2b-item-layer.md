# Codex Task: フェーズ2b 素体霊具レイヤー

前提資料(先に読むこと):

- RULE_SPEC.md(正本。§12 霊具)
- `docs/baselines/pure-vanilla-baseline-v3.md` — 対照群(凍結済み)

目的: デッキに霊具8枚を加えて「デッキの物理」を実ゲームに近づけ、**合格ゲート「平均ラウンド8.5〜10」**を判定する。素体霊具は影響クラスを代表するプレースホルダーであり、霊具そのもののデザインはフェーズ4で行う。

## ルール上の注意(毎回明示)

- 回転は式神1体につき1ターン1回まで(RULE_SPECの記載に関わらず裁定済みの例外)
- 撃破判定で撃破された式神は反撃しない
- 盤面から墓地へ送られた式神の所有者は `DESTROY_MANA_GAIN=1` を獲得(除去霊具によるキルでも同様)
- 霊具は霊力が続く限り回数無制限(§6.2)。ただし**召喚を行うとメインフェイズが終了する**ため、霊具は召喚前にしか使えない
- 使用した霊具は墓地へ(§12)。墓地はデッキ切れ時にリシャッフルされる

## 1. 素体霊具の定義(4原型)

| ID | 名前 | コスト | 効果 |
|---|---|---:|---|
| item_removal | 素体・除去 | 2 | 盤面の敵式神1体に2ダメージ(対象は全域から選択、反撃なし) |
| item_economy | 素体・経済 | 1 | 霊力を2得る(正味+1。MAX_MANA=15の上限は通常どおり) |
| item_buff | 素体・強化 | 2 | 味方式神1体のHPを+2する(永続。MAX_HP=10を超えない) |
| item_draw | 素体・ドロー | 1 | カードを1枚引く(手札上限超過はターン終了時に処理) |

デッキ構成(本命): 式神16枚(v3のlowmid分布)+ 霊具8枚 = **各原型2枚**、計24枚。

## 2. 霊具AI(最小仕様)

方針: 賢すぎる必要はないが、「持っているのに一切使わない」と計測が無意味になる。以下のヒューリスティックで実装:

- **除去**: (1) チェック返答探索とオラクルの行動空間に含める(王手返しの手段として正しく評価される事が最重要)。(2) 平時は「2ダメージで撃破が確定する敵がいる」かつ「使用後も予定の召喚が可能」な場合に使用
- **経済**: 使用によって「このターンに追加の行動(召喚・再命令)が可能になる」場合、または霊力が上限に遠く手札で腐っている場合に使用
- **強化**: 召喚予定を妨げない余剰霊力があり、味方式神が存在する場合に使用(対象は脅威にさらされている占拠要員を優先)
- **ドロー**: 余剰霊力1があり、手札に召喚可能な式神が少ない(2枚以下)場合に使用

実装の解釈で迷ったらシンプルな方を選び、実装ノートに記録すること。
