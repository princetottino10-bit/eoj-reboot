# 異能学園総選挙 — ゲームルール参照

ゲームエンジン実装のための機械可読ルール抜粋。  
全文は `docs/EOJR_Rulebook.md` を参照。

---

## 勝利条件

| 条件 | 内容 |
|---|---|
| VP勝利 | 15VP を先に獲得 |
| 領土勝利 | 自分のターン終了時に盤面5マス以上を占拠 |
| タイムアウト | 50ターン経過時にVPが多い方が勝利 |

### VP獲得（敵キャラ撃破時）

| 撃破した敵のコスト | 獲得VP |
|---|---|
| 1〜2 | 1VP |
| 3〜4 | 2VP |
| 5 | 3VP |

### マナ獲得（自軍キャラ撃破時）

自分のキャラが撃破された場合（攻撃・反撃問わず）、**撃破されたキャラのオーナーがマナ +1** を得る。

## ターン構造

```
① ターン開始
  - 全味方の「行動済み」「回転済み」フラグをリセット
  - 状態異常の継続ターンを1進める

② マナ獲得・ドロー
  - マナ +2（上限なし）
  - 山札から1枚ドロー

③ メインフェイズ（繰り返し可）
  以下を任意の順・回数で実行：
  - 召喚（キャラ1体）      → コスト: 召喚コスト、実行後ターン即終了
  - 再行動（攻撃 or 回転）  → コスト: 再行動コスト、ターン継続
  - アイテム使用           → コスト: アイテムコスト、ターン継続
  - ウルト発動             → コスト: VP消費、ターン継続（行動済み扱い）
  - パス                   → ターン終了

④ ターン終了処理（→「ターン終了」セクション参照）
```

---

## 召喚

1. 手札からキャラカード1枚を選ぶ
2. 召喚コスト分のマナを支払う
3. 盤面の空きマスに向きを選んで配置
4. 召喚時効果が発動
5. **自動攻撃**: 攻撃範囲に敵がいれば自動的に1回攻撃
6. **このターンは即終了**

### 配置制限

- 初回（盤面に味方0体）: どのマスでも可
- 2回目以降: **既存の味方または敵に隣接するマス（縦・横）のみ**
- 盤面上限: なし

### 属性マスによるHP補正（召喚時のみ）

| 状況 | HP補正 |
|---|---|
| キャラと同属性のマス | +2 |
| キャラと対立属性のマス | −2 |
| 虚マス または 属性不一致 | ±0 |

対立関係: 拳 ⇔ 念、光 ⇔ 闇（虚は中立）

---

## 再行動（攻撃・回転）

- **攻撃**: 再行動コスト分のマナを支払い、攻撃範囲内の敵1体を攻撃
- **回転**: 再行動コスト分のマナを支払い、キャラの向きを変える

### 制限

- 再行動による攻撃は同一キャラで同ターン中 **1回まで**（召喚時の自動攻撃はこの制限に含まない）
- 回転は攻撃とは独立に **1回まで**（同ターンに攻撃＋回転は両方可能）
- マナがあれば何体でも順次動かせる

### 再行動コスト

- カード記載の値（通常1〜3）
- `action_tax` 状態のキャラは追加コスト+N

## 戦闘

### 手順

1. 攻撃キャラを選択、再行動コストを支払う
2. 攻撃範囲内の敵1体を選択
3. B位置判定（攻撃側が守備側の死角にいるか）
4. 先制チェック
5. ダメージ計算・適用
6. HP0以下 → 撃破・VP獲得

### ダメージ計算

```
最終ダメージ = ATK
             + B位置ボーナス (+2)
             + 各種バフ
             − 防護 (−1、最低0)
             − チームDR
```

### B位置（ブラインドスポット）

攻撃側が守備側の死角（`weakness_cells`）にいる場合：
- ATK +2（**物理攻撃のみ**。魔法攻撃ではB位置ボーナスは発生しない）
- 守備側は反撃不可

### 反撃

守備側は以下の **全条件** を満たす場合、自動的に反撃を行う。

| 条件 | 内容 |
|---|---|
| 攻撃種別 | **物理攻撃のみ**（魔法攻撃では反撃しない） |
| 射程 | 攻撃者が **守備側の attack_cells 射程内** にいる |
| B位置優先 | 攻撃者が守備側の **weakness_cells（B位置）にいる場合は反撃不可**（射程内でも） |

その他の制約:
- 反撃に対して再反撃は発生しない
- 反撃はカバーされない（カバー役は反撃しない。ただし **要塞**持ちカバー役は例外）

### 先制（quickness）

- 守備側が先制を持つ場合、攻撃される前に反撃ダメージを与える
- B位置から攻撃された場合は先制も無効

### カバー

- カバー持ちキャラの**隣接味方**が攻撃を受けた場合、カバー持ちが代わりにダメージを受ける
- カバー発動時は原則として反撃発生なし
- 例外: カバー役が【要塞】を持つ場合、カバー役自身が反撃する

## キーワード一覧

### 防御系

| キーワード | 効果 |
|---|---|
| 防護 | 受けるダメージ−1（最低0）。永続（消費しない） |
| 回避 | 物理攻撃を1回無効化。マーカーとして消費される |
| 要塞 | 自ら攻撃できない。反撃のみ可能 |
| 不動 | 押し出し・引き寄せ効果で動かない |
| カバー | 隣接味方が受けるダメージを代わりに受ける |

### 攻撃系

| キーワード | 効果 |
|---|---|
| 貫通 | 攻撃時、防護・回避を無視 |
| 先制 | 反撃時、攻撃側より先にダメージを与える |

### マーカー（消費型バフ）

| マーカー | 効果 |
|---|---|
| 防護マーカー | 受けるダメージ−1（1回消費） |
| 回避マーカー | 物理攻撃を1回無効化（1回消費） |
| 貫通マーカー | 攻撃時、防護・回避を無視（1回消費） |
| 先制マーカー | 反撃時、先に攻撃（1回消費） |

キーワード持ちは永続、マーカーは1回使用で消える。  
判定上は両方とも同等に扱う（例: 「防護を持つ」= キーワードでもマーカーでも適用）。

## 状態異常

| 状態 | 効果 | 解除 |
|---|---|---|
| 洗脳（brainwashed） | 行動不能（攻撃・再行動不可） | duration終了 or 付与者の死亡 |
| action_tax | 再行動コスト +N | 付与者の死亡 |
| direction_lock | 回転不可 | duration終了 |
| atk_down | ATK −N | duration終了 or 永続 |
| 無敵（immune） | ダメージを受けない（反撃・召喚時攻撃含む） | duration終了 |

---

## チームダメージ軽減（teamDR）

特定のウルト効果で発動する全体バフ。

- 発動中、そのプレイヤーの**全キャラ**が受けるダメージが **−1**（最低0）
- 通常攻撃・反撃・召喚時攻撃すべてに適用
- **相手のターン終了時**に自動解除される（自分のターンの間のみ有効）
- `GameState.teamDR[player]` フラグで管理

---

## ターン終了処理

1. 領土勝利チェック（自分のターン終了時のみ）
2. ウルト効果クリーンアップ（相手の teamDR を解除）
3. 状態異常の duration −1
4. 手札上限チェック: **7枚**超過分はランダムに捨てる
5. ターンプレイヤー切替

## 派閥一覧

| ID | 名称 | 戦略 |
|---|---|---|
| aggro | アグロ（紅蓮） | 接敵・近距離・連撃 |
| tank | タンク（蒼盾） | 防御・カバー・領土勝利 |
| control | コントロール（紫闇） | 妨害・操作・洗脳 |
| synergy | シナジー（翠光） | 連携・マーカーバフ |
| snipe | スナイプ（銀眼） | 狙撃・遠距離 |
| trick | トリック（鉛影） | 撹乱・潜伏・位置入替 |

### エース回収条件（コスト5キャラのコスト軽減）

条件1つにつきコスト−1（最低2）。

| 派閥 | 軽減条件 |
|---|---|
| aggro | このターン攻撃した味方の数 |
| tank | カバー持ちで隣接味方がいるキャラの数 |
| control | デバフ状態（atk_down / action_tax / brainwashed）の敵の数 |
| synergy | マーカー持ちの味方の数 |
| snipe | 照準が付いた敵の数 |
| trick | 敵のB位置にいる味方の数 |
